React - Virtual DOM？

Virtual DOM 是一个轻量级的 JavaScript 对象，描述了真实 DOM 的层次结构。

在 React 中，当状态发生变化时，React 会创建一个新的 Virtual DOM 对象来代表新的 DOM 层次结构，并使用算法比较旧的和新的 Virtual DOM 对象之间的不同之处。根据 diff 算法的结果，React 只更新需要更改的部分，而不需要重新渲染整个页面。
