React - useMemo 和 useCallback 区别？

useMemo 和 useCallback 在缓存的对象和返回值上存在差异，但二者都可以优化 React 函数组件性能。

useMemo 可以根据依赖项的变化来缓存计算结果，从而避免不必要的重复计算。例如：

使用 useMemo 和 useCallback 都可以有效地避免在 React 应用程序中重复渲染组件，提高应用程序的性能和响应速度。
