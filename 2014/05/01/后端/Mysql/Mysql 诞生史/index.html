Mysql 诞生史

🪴

历史

MySQL 是一个开源的关系数据库管理系统，数据存放在表中。最初开发者为瑞典的 MySQL AB 公司，

1994 年，MySQL 的最初由 Widenius 和 Axmark 开始开发。

1995 年 5 月 23 日，MySQL 发布第一个版本。

2008 年，被升阳微系统收购。

2009 年，甲骨文公司收购升阳微系统公司，MySQL 成为 Oracle 旗下产品。
